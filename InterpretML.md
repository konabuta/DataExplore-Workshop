# Microsoft InterpretML

解釈可能な一般化加法モデルをベースにしたアルゴリズム。 

**GitHub Repo** : [Interpret ML - Alpha Release](https://github.com/microsoft/interpret)

モデル全体での説明変数の重要度 (グローバル) と個々の予測値に対する説明変数の重要度 (ローカル) の両方に対応している。
